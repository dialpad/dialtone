# Message Input Recipe

<Subtitle>
  Message Input recipe wraps the Rich Text Editor for uses cases like conversations and writing notes etc.
</Subtitle>

Message input recipe consists of the Rich text editor along with a container that houses the image picker,
emoji picker and slots for helper texts and image carousels.
It further has the logic to handle character limit logic and other message input functionality.
