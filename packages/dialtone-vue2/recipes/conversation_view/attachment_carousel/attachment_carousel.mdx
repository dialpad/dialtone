# DtRecipeAttachmentCarousel

<Subtitle>
  Attachment carousel is used in message input or any place where we need a
  preview of all uploaded files before its sent to the user in feed.
</Subtitle>

## Base Style

Attachment carousel can house mediaType of image/video for now.
It has the ability to preview each individual media and also remove them from the list.
