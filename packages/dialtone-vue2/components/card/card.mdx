# Card

<Subtitle>
    A card contains summary content and actions about a single subject.
    It can be used by itself or within a list, and is generally interactive.
</Subtitle>

## Base Style

Base card style with content.

<Canvas>

## Slots, Props & Events

<Controls />

## Usage

You can use the card with default content slot or combine header, footer for the complete look.

### Import

```jsx
import { DtCard } from '@dialpad/dialtone-vue';
```

## With header

A card with the header.

<Canvas>
    <Story of={CardStories.WithHeader} />
</Canvas>

```html
 <dt-card class="d-w264">
    <template #header>
      <p>Header</p>
      <dt-button
        size="xs"
        importance="clear"
        aria-label="Menu button"
      >
        <template #icon>
          <dt-icon name="more-vertical" />
        </template>
      </dt-button>
    </template>
    <template #content>
      <p class="d-fs-200 d-fw-bold">Lorem ipsum</p>
      <p class="d-fs-100">Lorem ipsum dolor sit amet, consectetur adipiscing elit.
      Donec fermentum molestie semper. Morbi finibus nulla turpis, nec molestie mi rutrum.</p>
    </template>
</dt-card>
```
