# Checkbox

<Subtitle>
  Checkboxes are control elements that allow the user to make a selection. They are typically used in a Checkbox Group
  which allows the user to make one or more selections from a list of options.
</Subtitle>

## Base Style

Checkboxes are typically paired with a label which identifies the option. In addition, there are times when they can be
used with a description which describes the option.

## Usage

### Import

```jsx
import { DtCheckbox, VALIDATION_MESSAGE_TYPES } from '@dialpad/dialtone-vue';
```

### With Label Prop

```jsx
<dt-checkbox
  value="optionValue"
  label="My Label"
/>
```

### With Label and Description Props

```jsx
<dt-checkbox
  value="optionValue"
  label="My Label"
  description="My Description"
/>
```

### With Slotted Label

```jsx
<dt-checkbox
  value="optionValue"
>
  My Label
</dt-checkbox>
```

### With Slotted Description

```html
<dt-checkbox
  value="optionValue"
  label="My Label"
>
  <template #description>
    My Description
  </template>
</dt-checkbox>
```

### With Slotted Label and Description

```html
<dt-checkbox
  value="optionValue"
>
  My Label
  <template #description>
    My Description
  </template>
</dt-checkbox>
```

### Checked

```jsx
<dt-checkbox
  value="optionValue"
  label="My Label"
  checked
/>
```

### Disabled

```jsx
<dt-checkbox
  value="optionValue"
  label="My Label"
  disabled
/>
```

### With Validation State

<h4 style={{ color: 'green' }}>Success</h4>

```html
<dt-checkbox
  value="optionValue"
  label="My Label"
  :validation-state="VALIDATION_MESSAGE_TYPES.SUCCESS"
/>
```

<h4 style={{ color: 'Goldenrod' }}>Warning</h4>

```html
<dt-checkbox
  value="optionValue"
  label="My Label"
  :validation-state="VALIDATION_MESSAGE_TYPES.WARNING"
/>
```

<h4 style={{ color: 'red' }}>Error</h4>

```html
<dt-checkbox
  value="optionValue"
  label="My Label"
  :validation-state="VALIDATION_MESSAGE_TYPES.ERROR"
/>
```
