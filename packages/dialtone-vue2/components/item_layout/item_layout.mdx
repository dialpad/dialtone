## Usage

### Import

```jsx
import { DtItemLayout } from '@dialpad/dialtone-vue';
```

### With default styling

By default, item layout includes custom styling, like paddings, sizes, colors, etc.

```html
<dt-item-layout />
```

### Without styling

Setting the `unstyled` property, removes all the custom styling maintaining the slots positions.
This way you can utilize the layout and customize your own styling.

```html
<dt-item-layout unstyled />
```
