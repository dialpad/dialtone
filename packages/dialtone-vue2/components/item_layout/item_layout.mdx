## Usage

### Import

```jsx
import { DtItemLayout } from '@dialpad/dialtone-vue';
```

### With default styling

By default, item layout includes custom styling, like paddings, sizes, colors, etc.

```html
<dt-item-layout />
```

### Without styling

Setting the `unstyled` property will add `dt-item-layout--custom` class.
This will change the item-layout from flexbox to grid,
removing all the custom styling while maintaining the slots positions.

This way you can utilize the layout and customize your own styling using utility classes.

```html
<dt-item-layout unstyled />
```
