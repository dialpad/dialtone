## Sizes

Dialtone provides eight sizes for icons, three of them are **device only**. Each of the sizes represents the width and
a height the icon is going to have:

1. **100**: 12px (1.2rem)
2. **200**: 14px (1.4rem)
3. **300**: 18px (1.8rem)
4. **400**: 20px (2.0rem)
5. **500**: 24px (2.4rem)
6. **600**: 32px (3.2rem)
7. **700**: 38px (3.8rem)
8. **800**: 48px (4.8rem)

> **Please note** These are the only available size options and no overrides should be needed to properly size the icons.

## Accessibility

If you use an icon to portray something more than just it's visual appearance, make sure to add a description of its
function to the ariaLabel prop.

Example: A down arrow icon button that opens a dropdown settings menu, set the ariaLabel as 'open settings menu'.
